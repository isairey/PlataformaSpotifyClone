# 🌌 Acerca de Streamify

**Streamify** es un clon de Spotify enfocado en ofrecer una experiencia musical moderna mediante tecnologías web como TypeScript y JavaScript, incorporando reproducción multimedia, playlists dinámicas y una interfaz inspirada en plataformas de streaming profesionales.

La aplicación permite:

- 🎵 Escuchar música online
- 🔎 Buscar canciones y artistas
- ❤️ Gestionar favoritos
- 📚 Crear playlists
- 🎧 Reproducir audio en tiempo real
- 🌙 Dark Mode moderno
- ☁️ Streaming multimedia avanzado

El proyecto está orientado al aprendizaje y práctica de:

- TypeScript
- Frontend Development
- Multimedia Applications
- REST APIs
- Audio Streaming
- UI/UX moderna
- Arquitectura web

# ⚡ Instalación

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaSpotifyClone
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaSpotifyClone
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar entorno desarrollo

```bash
npm run dev
```

---

## 5️⃣ Compilar proyecto

```bash
npm run build
```
